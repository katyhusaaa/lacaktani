Dokumentasi LacakTani.ai

Selamat datang di panduan resmi LacakTani.ai. Proyek ini dirancang untuk mendeteksi fase pertumbuhan stroberi secara real-time menggunakan computer vision (YOLOv5) yang diintegrasikan dengan UAV (Drone).

💡
LacakTani adalah proyek Open Source. Kamu bisa menemukan source code lengkap di GitHub @agungxsoeltani.

System Requirements

Sebelum memulai instalasi, pastikan lingkungan pengembangan kamu memenuhi kriteria berikut:

  • Python 3.10+ (Wajib)
  • CUDA Toolkit (Opsional, untuk akselerasi GPU)
  • ESP32-CAM dengan firmware terbaru
  • RAM Minimal 8GB

Quickstart Installation

Jalankan perintah berikut di terminal kamu untuk mengunduh dan menginstal seluruh dependensi secara otomatis:

git clone https://github.com/agungxsoeltani/lacaktani.git
cd lacaktani
pip install -r requirements.txt
python app.py
⚠️
Jika kamu menggunakan Windows, sangat disarankan menjalankan sistem di dalam WSL2 atau virtual environment (venv) untuk menghindari konflik pustaka PyTorch.

Hardware Configuration

Modul ESP32-CAM harus dikonfigurasi untuk memancarkan stream HTTP di alamat lokal. Pastikan file config.h di Arduino IDE sudah diset ke mode VGA (640x480) agar mendapatkan FPS maksimal.

// Arduino IDE Setup
#define CAMERA_MODEL_AI_THINKER
#define FRAMESIZE_VGA  // Set ke VGA untuk FPS stabil

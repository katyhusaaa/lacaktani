Arsitektur Alur Kerja

Deployment didesain sesederhana mungkin. Tidak perlu instalasi berat, langsung terbang dan lakukan deteksi.

01

Deploy Hardware

Terbangkan drone yang telah dilengkapi ESP32-CAM atau hubungkan dengan sumber kamera lokal di kebun.

02

Establish Link

Masukkan IP Address dari Stream video ke dalam Dashboard Web untuk mengunci koneksi dengan server.

03

AI Inference

Mesin YOLOv5 memproses aliran video secara real-time, mendeteksi fase komoditas dengan presisi tinggi.

04

Generate Report

Akhiri sesi penerbangan. Sistem mengkompilasi data menjadi estimasi hasil panen di menu Riwayat.

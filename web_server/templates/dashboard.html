Panduan Operasional

Langkah cepat kalibrasi sistem LacakTani.ai

LANGKAH 1

Setup Koneksi Video

Klik tombol Konek Kamera di pojok kiri atas. Pilih ESP32-CAM jika ingin memantau via *drone*, atau Webcam untuk *testing* lokal. Pastikan IP terisi benar jika menggunakan ESP32.

LANGKAH 2

Kalibrasi Parameter

Buka menu Settings. Atur resolusi (disarankan VGA 640x480 agar FPS stabil) dan atur Confidence Threshold AI YOLOv5. Angka ideal untuk deteksi stroberi adalah ~0.40 - 0.50.

LANGKAH 3

Mode Eksekusi AI

Gunakan tombol Uji Coba AI (hitam) hanya untuk sekadar memantau tanpa menyimpan data. Klik Mulai Sesi Patroli (merah) jika Anda ingin hasil kalkulasi direkam dan masuk ke Database Laporan.

Pro Tip: Jika terjadi False Positive (AI salah mendeteksi objek), Anda dapat menekan ikon *refresh* di sebelah angka atau tombol Reset Angka ke 0 untuk membersihkan kalkulasi secara manual.
